v0.3. Two modes share one desktop companion. Tap Ctrl+Space and ask any question — get a short spoken answer back via Ava (Premium) in ~1-1.5s, with the reply text in a floating top-right note. Tap Ctrl+Shift+Space instead to spawn an autonomous sandboxed claude -p agent that runs a real task to completion with a status puck.

Quick-ask features: conversational follow-ups in a 60s window · semantic style preferences ("be shorter", "more detail" — no keyword matching) · interrupt mid-speech by tapping Ctrl+Space again · pluggable backends (slow default, fast API-key, or custom) · pre-warmed startup. A mystical feather animates next to the answer note as a state indicator — color shifts violet → pink → gold → mint through idle/listening/thinking/speaking.

The walkthrough below simulates the v0.1 agent-spawn flow. Quick-ask is best experienced on your own machine — clone, install Ava (Premium), tap Ctrl+Space.

// Real process control

Pause via SIGSTOP to the agent process group. Cancel via SIGTERM → SIGKILL. Amend queues a follow-up via claude --continue. Not faked.

// Cross-app overlays

Custom PyObjC shim pins Qt windows at NSStatusWindowLevel with canJoinAllSpaces. The pucks survive Mission Control, Spaces switches, and full-screen apps.

// Streaming everything

STT chunks drive the reactive cursor indicator. Agent stdout parsed live as stream-json so the puck reflects what the agent is doing — not just "running".
